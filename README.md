 ## How To Setup Step 1

 anda dapat mengunduh projectnya di branch <b>" save/load-user-progress-(local)-(step-1) "</b>

 Note : jangan lupa ekstrak dulu file <b>" Firebase.rar "</b> di dalam folder Asset sebelum open projectnya.
